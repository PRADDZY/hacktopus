# Frontend Workspace

This folder contains both deployable frontend applications:

- `shop/` -> customer checkout app
- `dashboard/` -> bank/admin analytics dashboard

## Shop

```bash
cd frontend/shop
npm install
npm run dev
```

## Dashboard

```bash
cd frontend/dashboard
npm install
npm run dev
```

Both apps use:

- `NEXT_PUBLIC_BACKEND_URL` in `.env.example`
